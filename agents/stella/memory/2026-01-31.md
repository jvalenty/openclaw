### Agent Type Architecture Decision
After discussion, settled on:
- `sys_admin` - Machine admin (1 per machine, full system access)
- `manager` - Coordinates and routes tasks to other agents
- `specialist` - Domain experts (trading, coding, research)
- `assistant` - General purpose helpers

Key insight: sys_admin is the software+hardware layer admin for a machine.
