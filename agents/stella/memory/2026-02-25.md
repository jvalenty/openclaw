### Docker Fix (Important Lesson)
When server imports from root src/ (backtest engine imports chalk, drizzle, etc.):
- Solution: COPY the specific files into server/src/lib/ and strip external deps
- DON'T: include root src/** in server tsconfig (pulls in too many unresolvable deps)
- DON'T: install root node_modules in Docker (heavyweight, error-prone)
- chalk v5 is ESM-only — if server uses CommonJS, use chalk v4

### Backtest API Contract
