## HMAC Token Note
Tokens derived from `INTERNAL_SERVICE_TOKEN` Fly secret via `hmac(fileId, secret).slice(0,32)`.
If `INTERNAL_SERVICE_TOKEN` is rotated, all existing signed URLs in DB metadata become invalid.
Patched messages would need re-signing. Not urgent — only rotate if security requires it.
