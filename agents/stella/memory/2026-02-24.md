## Agent Learning System (Bella's design — EXCELLENT)
`src/db/learning-schema.ts` — full recursive learning architecture:
- `strategy_pool`: validated shared strategies (seeded with TSLA ORB 30m 2x)
- `agent_variants`: per-agent param overrides with performance tracking
- `trade_log`: full params on every trade for retroactive analysis
- `performance_stats`: daily rollup per agent × variant
- `pool_sync_events`: audit trail for promoting winning variants to pool
- `model_id` column: tracks which LLM model each agent uses (for attribution!)

**Circuit breakers:**
- 3 consecutive losses/day → day_break (halt until tomorrow open)
- 3 consecutive losing days → extended_break (3-day cooldown)
- Winning days: +20% allocation (measured, not binary)
- Losing days: -30% allocation (asymmetric — cut losers faster)
- Floor: $1K (agent stays alive), Ceiling: $100K
