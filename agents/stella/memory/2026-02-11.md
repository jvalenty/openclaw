## Architecture Decisions

### Org vs Team Scoping
- **Orgs** = Hard security boundary (different customers)
  - Machine Service enforces at infrastructure level
  - Separate browser profiles, file sandboxes
- **Teams** = Soft visibility boundary (same company, departments)
  - Stellabot enforces at application level
  - Controls what agents/data users see within org

### Google Credential Hierarchy
1. Check `agent_secrets` for agent-specific OAuth
2. Fall back to `integration_credentials` (org-level)
3. Error if neither exists
