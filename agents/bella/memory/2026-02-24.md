### Key Systems
**TradingEngine pipeline:**
Polygon bar → OrbVolumeStrategy → AllocationManager.canTrade() → CatalystPlanner.canTrade() → PaperEngine → DBBridge → Neon

**AllocationManager rules (John's spec):**
- 3 consecutive losses/day → day_break
- 3 consecutive losing days → extended_break (3-day cooldown, allocation halved)
- Winning day: +20% allocation; Losing day: -30% (asymmetric)
- Floor: $1K, Ceiling: $100K

**CatalystPlanner (3 layers):**
1. Scheduled: earnings (block), FOMC (×0.4), CPI/Jobs (×0.5), options expiry (×0.7)
2. Live news: Polygon WebSocket + REST, sentiment scoring, breaking news detection
3. Historical: pattern mining from trade_log (placeholder → fills over time)
Additional: market hours enforcement, lunch window, late session block

**Learning schema:**
- strategy_pool: shared validated strategies (baseline)
- agent_variants: per-agent param tweaks with rolling perf tracking
- trade_log: full params snapshot on every trade (retroactive analysis)
- pool_sync_events: audit trail when variant beats baseline → promoted
