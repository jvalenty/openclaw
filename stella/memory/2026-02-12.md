## Late Night Session (~9:45 PM PST) — Bug Fixes

### Thread Bleed Fix
Race condition in GlobalChatSidebar.tsx where switching agents caused stale thread messages to sync to wrong agent's view.

**Root cause:** useEffect syncing threadMessages didn't guard against agentId mismatch after tab switch.

**Fix:** Added `threadSession?.agentId === activeTab?.agentId` guard to sync effect.

Commit: `7c6eea4` — deployed to Fly.io

### Stella Soft Disappeared
John deleted Stella Hard, renamed Stella Soft to "Stella", changed type to sys_admin → she vanished from UI.

**Root cause:** `sys_admin` type without `machineId` or `endpoint` = invalid hard agent, filtered out.

**Fix:** Changed her back to `manager` type via SQL.

**Lesson:** Agent types determine routing:
- `sys_admin` = hard agent (needs machineId/endpoint)
- `manager`, `specialist`, etc. = soft agent (Claude API)

Capabilities come from `agent_actions` permissions, not agent type.
