### Key Learnings / Gotchas
1. **OpenClaw not clawdbot** — npm package renamed to `openclaw` (github.com/openclaw/openclaw)
2. **Tailscale Serve requires `bind: loopback`** — can't use `bind: lan`
3. **Tailscale Serve --bg flag** — use `sudo tailscale serve --bg 18799` for persistent background mode (not `--set-path`)
4. **Device pairing** — new browsers need approval via `openclaw devices approve <request-id>`
5. **Device token mismatch** — clear `~/.openclaw/identity` and `~/.openclaw/devices` if stuck, restart gateway
6. **No Homebrew in Ubuntu** — install skills deps via apt instead
7. **Config format changed** — `agent.model` → `agents.defaults.model.primary`
8. **allowInsecureAuth** — needed if accessing Control UI without proper device identity
