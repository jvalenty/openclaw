## Agent Actions Org Membership Fix (~2:43 PM PST)

### Problem
John saw "Failed to Fetch Actions" when viewing Dan's agent_actions in Earnware org settings.

### Root Cause
`agent-actions.ts` used `user.orgId` (the user's default org from session) to verify access. John's default org was AMG (`fb6ec0c9-...`), but Dan is in Earnware (`22377b4f-...`). Even though John was viewing Earnware settings, his session org was still AMG.

### Fix
Changed agent-actions API to check org membership instead of default org:
1. Get the agent first
2. Check if user is a member of the agent's org via `orgs_members` table
3. Allow access if membership exists

**Commit:** ca82c51 "Fix agent-actions to check org membership instead of default org"

### Key Change
```javascript
// OLD: Used user's default org
const orgId = getOrgId(req); // user's default org
eq(agents.orgId, orgId) // fails if agent in different org

// NEW: Check membership in agent's org
const agent = await db.query.agents.findFirst({ where: eq(agents.id, agentId) });
const isMember = await isOrgMember(userId, agent.orgId);
```

### Lesson
Multi-org users need membership-based access checks, not default-org checks.

## Sheets AgentId Fix (~2:49 PM PST)

### Problem
Dan's Sheets tools were returning "permission denied" тАФ he couldn't read or write to Google Sheets.

### Root Cause
Sheets API endpoints (`/sheets/read`, `/sheets/write`, etc.) didn't pass `agentId` to the service functions. Dan has his own Google OAuth stored in `agent_secrets` (for his Google account), but the code was only looking for org-level credentials in `integration_credentials`. Earnware org doesn't have org-level Google OAuth set up.

### Fix
Updated all Sheets service functions and routes to accept and pass `agentId`:
- `getSheetsClient(orgId, agentId)` тАФ now checks agent_secrets first
- `getDriveClient(orgId, agentId)` тАФ same
- All Sheets endpoints now pass `req.agentId` to the service functions

**Commit:** 77ec029 "Pass agentId to Sheets service for agent-level OAuth"

### Key Change
```javascript
// OLD: Only checked org-level credentials
const client = await getSheetsClient(orgId);

// NEW: Check agent credentials first, fall back to org
const client = await getSheetsClient(orgId, req.agentId);
```

### Result
Dan can now use Google Sheets with his own OAuth credentials stored in agent_secrets.
